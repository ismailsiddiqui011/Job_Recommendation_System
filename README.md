# Job_Recommendation_System
## Libraries Required:
- Pandas
- Numpy
- Sklrean
- Operator
- Flask

## Software/Tool Used:
- Postman

How to Use:
### This is very simple to use app:
- Run the app.py or app.ipynb
- Enter the web address in web browser
- Input the User ID in Text Field
- Jackpot!! You got Job Recommendations
